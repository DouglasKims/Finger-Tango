Renpy files for "Finger Tango"

"script.rpy" is the default file that calls the "game screen" screen, a basic screen that allows selection of games.
"gamescreens.rpy" is a simple list of games within a "screen", a renpy statement. This is a good example of how using screens inside ren'py.

"puzzlegame_fingertango.rpy" contains the core and bulk of the game, it's a mix of python code and ren'py code, using heavily screen statements to add functionality to the game.

Ren'py Documentation for Screens:  https://www.renpy.org/doc/html/screens.html
